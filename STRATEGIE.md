## 1. MERKANALYSE & POSITIONERING

### Wat is Mokka Home Interior?

Mokka Home Interior is een **premium online interieurwinkel** gericht op het middenhoge tot hoge segment. Het merk verkoopt zorgvuldig gecureerde woonproducten: meubels, verlichting, decoratie en accessoires.

Het merk onderscheidt zich door:
- **Curation over quantity** — niet alles verkopen, maar het beste selecteren
- **Visuele storytelling** — producten worden gepresenteerd in context, niet als losse items
- **Toegankelijke luxe** — premium kwaliteit zonder onbereikbaar te zijn

### Doelgroep

**Primair:**
- Vrouwen en mannen, 28-50 jaar
- Bovenmodaal inkomen (€45K-€120K+)
- Woonachtig in Nederland en België (later: DACH)
- Interesse in interieurdesign, architectuur, lifestyle
- Volgen accounts op Instagram/Pinterest over interieurs
- Waarderen kwaliteit boven kwantiteit

**Psychografie:**
- Zoeken rust en esthetiek in hun leefomgeving
- Willen een "afgemaakt" interieur zonder zelf ontwerper te zijn
- Bereid meer te betalen voor iets dat mooi EN goed gemaakt is
- Vertrouwen merken die authentiek en transparant communiceren

**Koopgedrag:**
- Onderzoeken online, kopen online of na showroombezoek
- Gemiddelde orderwaarde: €150-€800
- Kopen 2-4x per jaar interieurproducten
- Laten zich inspireren door "shop the look" en sfeerbeelden

### Concurrentieanalyse

| Concurrent | Stijl | Prijsklasse | Zwakte |
|-----------|-------|-------------|--------|
| Westwing | Trendy, sales-gedreven | Midden | Druk, veel kortingen = goedkoop gevoel |
| Flinders | Design-georiënteerd | Hoog | Klinisch, weinig sfeer |
| WOOOD | Scandinavisch budget | Laag-midden | Niet premium genoeg |
| Tine K Home | Boho-luxe | Midden-hoog | Niche, beperkt assortiment |
| Audo (Menu) | Ultra-premium | Zeer hoog | Onbereikbaar voor meeste klanten |

**Mokka Home Interior positioneert zich tussen Flinders en Audo:**
Premium kwaliteit, industrieel-warme uitstraling, ambachtelijk gevoel.
Toegankelijker dan ultra-luxe maar duidelijk boven massamarkt.
Denk aan: Belgisch design, wabi-sabi, Axel Vervoordt, raw luxury.

### Merkpersoonlijkheid

Afgeleid van het logo: gehamerd ijzer = ambacht, kracht, authenticiteit.

| Eigenschap | Betekenis |
|-----------|-----------|
| **Ambachtelijk** | Eerlijke materialen, zichtbaar vakmanschap, niet massaproductie |
| **Rauw & Warm** | De kracht van ijzer gecombineerd met de warmte van een thuis |
| **Verfijnd** | Oog voor detail, niets is willekeurig, elk stuk is doordacht |
| **Rustig** | Geen schreeuwende banners — vertrouwen uitstralen, niet pushen |
| **Tijdloos** | Geen trends najagen, investeren in stukken die blijven |

### Tone of Voice

- **Taal:** Nederlands (primair), later Engels
- **Stijl:** Kalm, persoonlijk maar professioneel
- **Niet:** "SALE!", "OP=OP!", "Bestel NU!"
- **Wel:** "Ontdek onze nieuwe collectie", "Zorgvuldig geselecteerd", "Gemaakt om bij te blijven"
- **Formeel niveau:** Je-vorm (niet u), maar wel respectvol
- **Voorbeeld:** "Elk stuk in onze collectie is geselecteerd met één vraag in gedachten: maakt dit je huis mooier om in te leven?"

## 2. VISUELE IDENTITEIT

### Logo

**Bestaand logo:** "MOKKA HOME INTERIOR" in gehamerd ijzer textuur.

**Kenmerken:**
- **"MOKKA"** groot en dominant, **"HOME INTERIOR"** kleiner eronder
- **Textuur:** Gehamerd/gesmeed ijzer — ambachtelijk, tactiel, rauw
- **Kleur:** Donker gunmetal/ijzer (#3D3D3D met structuur)
- **Lettertype:** Decoratief serif met organische curves
- **Stijl:** Industriële luxe, artisanaal, natuurlijke materialen

**Varianten nodig voor web:**
- **Volledig logo:** PNG transparant (voor homepage, footer)
- **Compact:** "MOKKA" alleen of "M" icoon (voor navbar, favicon)
- **Donker op licht** (primair — logo op cream/witte achtergrond)
- **Licht op donker** (voor donkere secties — logo in licht ijzer/zilver)
- **Favicon:** Letter "M" in de ijzer-stijl, 32x32 en 180x180

**Benodigde bestanden:**
- PNG transparant in hoge resolutie (minimaal 2000px breed)
- Optioneel: SVG versie (voor schaalbaar gebruik)
- Favicon versies (ICO + PNG)

### Kleurenpalet

Afgeleid van het gehamerd ijzer logo. Het palet combineert de rauwe,
industriële kracht van ijzer met warme, aardse tinten voor balans.

```
PRIMAIR (afgeleid van het logo)
├── Iron         #3D3D3D  — Logo kleur, koppen, primaire knoppen
├── Warm White   #F8F6F3  — Hoofdachtergrond, witruimte, rust
└── Pure White   #FFFFFF  — Cards, productafbeeldingen, secties

ACCENT
├── Aged Bronze  #7A6652  — CTA's, prijzen, links, hover states
└── Warm Copper  #A0836C  — Subtiele highlights, iconen, actieve states

NEUTRAAL
├── Stone        #D4CFC9  — Borders, dividers, achtergrond variatie
├── Iron Gray    #6B6B6B  — Secundaire tekst, labels, placeholders
├── Ash          #9C9690  — Tertiaire tekst, meta-informatie
└── Soft Gray    #F0EDEA  — Hover achtergronden, skeleton loading

DONKERE SECTIES (voor contrast-blokken, footer, hero)
├── Dark Iron    #1C1C1C  — Donkere achtergronden
├── Anthracite   #2A2A2A  — Donkere cards, footer
└── Iron Light   #B0B0B0  — Tekst op donkere achtergrond

FEEDBACK
├── Succes       #4A7C59  — Bevestigingen, voorraad
├── Error        #B54A4A  — Fouten, waarschuwingen
└── Info         #5B7FA5  — Informatie
```

**Design Principes:**
- Warm White is de dominante kleur (60%+) — veel rust en witruimte
- Iron (#3D3D3D) voor tekst, koppen en primaire elementen (25%)
- Aged Bronze voor accenten, CTA's en interactieve elementen (10%)
- Stone voor structuur: borders, dividers (5%)
- Donkere secties spaarzaam maar impactvol gebruiken (hero, footer, merkblok)
- Nooit meer dan 3 kleuren tegelijk op één scherm
- Geen felle kleuren, geen gradients op knoppen
- Het ijzer-thema doorvertalen: rauwe texturen, natuurlijke materialen

### Typografie

**Primair (koppen):**
- **Font:** Playfair Display
- **Gebruik:** H1, H2, H3, productnamen, quotes
- **Stijl:** Regular + Italic (italic voor accenten)
- **Waarom:** Klassiek, elegant, leesbaar, gratis (Google Fonts)

**Secundair (body):**
- **Font:** Inter
- **Gebruik:** Paragrafen, labels, knoppen, navigatie
- **Stijl:** Light (300), Regular (400), Medium (500)
- **Waarom:** Modern, uitstekende leesbaarheid, variabele font

**Hiërarchie:**
```
H1:  48-80px  Playfair Display Regular    (hero titels)
H2:  36-56px  Playfair Display Regular    (sectietitels)
H3:  24-32px  Playfair Display Regular    (subsecties)
H4:  18-20px  Inter Medium                (card titels)
Body: 15-16px Inter Regular, line-height 1.7
Small: 12-13px Inter Regular              (labels, meta)
Caps:  11px   Inter Medium, tracking 0.2em (subtitels, navigatie)
```

### Fotografie Stijl

**Dit is CRUCIAAL voor een premium interieurmerk. Slechte foto's = geen verkoop.**

**Productfoto's (verplicht per product):**
1. **Hero shot** — Product in een gestileerde setting (kamer, tafel, etc.)
2. **Clean shot** — Product op witte/neutrale achtergrond
3. **Detail shot** — Close-up van materiaal, textuur, afwerking
4. **Scale shot** — Product in context zodat je de grootte begrijpt
- Minimum: 3 foto's per product
- Ideaal: 5-8 foto's per product
- Resolutie: minimaal 2000x2000px
- Format: WebP (met JPEG fallback)
- Achtergrond: neutraal, warm licht, geen harde schaduwen

**Sfeerbeelden (voor homepage, categorieën, "shop the look"):**
- Interieurscènes met meerdere producten
- Warm, natuurlijk licht — maar met schaduw en diepte (niet overbelicht)
- Rauwe materialen zichtbaar: hout, steen, beton, linnen, ijzer
- Textuur is belangrijk: je moet materialen bijna kunnen voelen
- Lifestyle element (een kop koffie, een boek, een kaars, een plant)
- Kleurenpalet: aardtinten, grijs, warm wit, zwart accenten
- Nooit: stockfoto-achtig, te kleurrijk, te perfect/gelikt

**Waar foto's vandaan halen (start):**
- **Optie 1:** Producten inkopen en zelf fotograferen (beste kwaliteit)
- **Optie 2:** Leveranciers vragen om hoge-resolutie productfoto's
- **Optie 3:** Start met Unsplash/Pexels voor sfeerbeelden (tijdelijk)
- **Optie 4:** AI-gegenereerde interieurscènes (kan als placeholder)

⚠️ **BELANGRIJK:** Gebruik NOOIT stockfoto's als productfoto's. Klanten verwachten het echte product te zien.

### Iconografie

- **Stijl:** Lijnpictogrammen, 1.5px dikte, afgeronde hoeken
- **Consistentie:** Eén icoonset door de hele site (Lucide Icons aanbevolen)
- **Kleur:** Warm Gray voor standaard, Charcoal voor actief/hover
- **Grootte:** 18-20px in navigatie, 24px in features, 48px in USP's

### Animaties & Interactie

**Principes:**
- Animaties ondersteunen de ervaring, ze zijn niet de ervaring
- Alles voelt vloeiend, niets springt of knippert
- Snelheid: 300-500ms voor de meeste transities
- Easing: cubic-bezier(0.25, 0.4, 0.25, 1) — zacht en natuurlijk

**Waar wel:**
- Scroll-triggered fade-in op secties (subtiel, 1x)
- Hover effecten op productkaarten (lichte lift + tweede afbeelding)
- Paginatransities (zachte fade)
- Winkelwagen notificatie (bounce-in)

**Waar NIET:**
- Geen parallax die de hele pagina beweegt
- Geen 3D scenes die de laadtijd verdubbelen
- Geen custom cursor (accessibiliteitsprobleem)
- Geen animaties die content blokkeren
- Geen animaties op mobiel die performance kosten

### Visuele Content (Instagram / Pinterest)

**Instagram:**
- Sfeerbeelden van producten in interieurs
- "Behind the scenes" van ontwerpers/ambachtslieden
- Klantfoto's (user generated content)
- Reels met styling tips
- Consistent kleurenpalet in feed (warm, neutraal)

**Pinterest:**
- Productpinnen met directe links naar shop
- Moodboards per stijl (Scandinavisch, Modern, Bohemian)
- "Shop the Look" pins
- Infographics (interieur tips)

### E-mail Marketing Flows

```
1. WELKOM FLOW (na registratie)
   → Direct: Welkomstmail + 10% korting eerste bestelling
   → Dag 3: "Ontdek onze collectie" met bestsellers
   → Dag 7: "Onze favorieten voor jou" met personalisatie

2. VERLATEN WINKELWAGEN (na 1 uur)
   → 1 uur: "Je hebt iets achtergelaten"
   → 24 uur: Herinnering met productafbeeldingen
   → 72 uur: Laatste herinnering (optioneel: gratis verzending)

3. POST-AANKOOP
   → Direct: Orderbevestiging
   → Bij verzending: Tracking informatie
   → 7 dagen na levering: "Hoe bevalt je aankoop?" + review verzoek
   → 30 dagen: Cross-sell suggesties

4. NIEUWSBRIEF (maandelijks)
   → Nieuwe producten
   → Inspiratie artikel
   → Eventuele actie

5. WIN-BACK (na 90 dagen geen activiteit)
   → "We missen je" + nieuwe collectie
   → Persoonlijke aanbevelingen
```

### Advertentiestrategie (Basis)

**Google Ads:**
- Shopping ads voor productfeed

- Display retargeting voor bezoekers

**Meta Ads (Instagram/Facebook):**
- Sfeerbeelden voor bereik
- Productcarousel voor conversie
- Retargeting op websitebezoekers en cart-abandoners

### Authenticatie
- NextAuth.js v5 met Prisma Adapter
- Sessies opgeslagen in database (niet JWT)
- Wachtwoord: bcrypt hash, min 8 tekens, complexiteitseisen
- Optioneel: Google/Apple OAuth
- Rate limiting op login: 5 pogingen per 15 minuten
- Account lockout na 10 mislukte pogingen

### Autorisatie
- Role-based: `customer`, `admin`
- Middleware beschermt `/admin/*` routes
- API routes controleren rol server-side
- Geen client-side security beslissingen

### Data Beveiliging
- Alle input gevalideerd met Zod (server-side)
- Prisma parameterized queries (geen SQL injection)
- XSS preventie via React's default escaping
- CSRF via SameSite cookies + Origin header check
- Geen gevoelige data in client-side code

### HTTP Beveiliging
```
Strict-Transport-Security: max-age=63072000; includeSubDomains; preload
X-Frame-Options: DENY
X-Content-Type-Options: nosniff
Referrer-Policy: strict-origin-when-cross-origin
Permissions-Policy: camera=(), microphone=(), geolocation=()
Content-Security-Policy: [configureren per domein]
```

### Betalingsbeveiliging
- Stripe handelt alle kaartgegevens af (PCI DSS compliant)
- Geen kaartdata raakt onze server
- Webhook signatures geverifieerd
- Idempotency keys voor dubbele betalingen
- Bedragen altijd server-side berekend (nooit client-side vertrouwen)

### API Beveiliging
- Rate limiting via Upstash Redis
- Server-side validatie op alle endpoints
- Geen stack traces in productie errors
- Environment variables voor alle secrets
- Regelmatige dependency audits

## 8. BETALINGEN ARCHITECTUUR

### Flow

```
Klant vult winkelwagen → Checkout pagina → Vult gegevens in
→ Klikt "Afrekenen" → Server maakt Stripe Checkout Session
→ Redirect naar Stripe (iDEAL / Card / Bancontact)
→ Betaling succesvol → Stripe webhook → Order aangemaakt in DB
→ Bevestigingsmail verstuurd → Klant ziet succespagina
```

### Betaalmethoden
- **iDEAL** (verplicht voor NL markt, ~60% van online betalingen)
- **Creditcard** (Visa, Mastercard)
- **Bancontact** (voor België)
- **Apple Pay / Google Pay** (via Stripe)
- **Klarna** (optioneel, voor "betaal later")

### Order Statussen
```
PENDING      → Bestelling aangemaakt, wacht op betaling
PAID         → Betaling ontvangen via Stripe webhook
PROCESSING   → In behandeling / wordt ingepakt
SHIPPED      → Verzonden, tracking beschikbaar
DELIVERED    → Afgeleverd
CANCELLED    → Geannuleerd (voor verzending)
REFUNDED     → Terugbetaald
```
